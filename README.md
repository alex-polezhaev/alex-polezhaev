### 👋 What I build

I build **complete, production-grade products - solo, from firmware to frontend.**
Native iOS, Swift &amp; Node backends, React/Next.js web, even embedded C++ firmware - designed and shipped end to end.

**8 years · 30+ projects shipped · working US Eastern hours · open to contract &amp; project work.**

- 🧩 **Web apps, end-to-end** - React / Next.js on a clean, scalable backend
- 📱 **iOS apps** - native SwiftUI + Swift backend, shipped to the App Store
- 🛠️ **Backends, APIs &amp; systems** - PostgreSQL, gRPC, queues, real-time streams
- 🔄 **E-commerce &amp; marketplace sync** - catalog, inventory &amp; orders, fully automated
- ⚡ **Automation &amp; internal tools** - n8n, Apps Script, custom internal panels
- 🤖 **AI features &amp; agents** - GPT pipelines, voice / transcription, custom agents
